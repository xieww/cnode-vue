###安装

项目地址：（`git clone`）

```shell
git clone https://github.com/xieww/cnode-vue.git
```

通过`npm`安装本地服务以及第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))

```国内npm比较慢，可以使用国内淘宝的cnpm(https://npm.taobao.org/)
